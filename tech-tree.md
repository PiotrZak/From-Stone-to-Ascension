🌌 CIVILIZATION TECH UNIVERSE TREE (CTUT)
🧠 DESIGN PRINCIPLE

Instead of a flat tree:

Each TTS is a “tech universe layer”
Each layer contains:
Core tech (mandatory progression)
Branch tech (optional specialization)
Forbidden tech (risk systems)
Fusion tech (cross-layer unlocks)

TTS1 → TTS2 → TTS3 → TTS4 → TTS5 → TTS6 → TTS7 → TTS8 → TTS9+

Each layer has ~20–60 nodes.

🔗 FUSION SYSTEM (KEY TO 500+ NODES)

Instead of manually adding nodes, expand using fusion rules:

Example Fusion Nodes:
AI + Genetics → Synthetic Organisms (TTS5–6)
Quantum + Time → Probability Editing Systems (TTS5–7)
Nano + Industry → Self-Building Civilizations (TTS6–8)
AI + Reality → Law-of-Physics AI (TTS8+)

Each fusion = +3 to +12 new nodes automatically.

⚙️ PROCEDURAL EXPANSION RULE (IMPORTANT)

To grow this into 500 nodes:

For each node define:

Node = {
  tech_type,
  tier (TTS level),
  prerequisites (1–3 nodes),
  risk_level,
  fusion_tags
}

Then generate:

3 child nodes per basic tech
5–8 child nodes per advanced tech
2 fusion nodes per pair of tags
